# Valid Chord Names

All chord names that Seqular recognizes. Use these exact strings in the `chords` array.

## Major
`C`, `C#`, `D`, `D#`, `E`, `F`, `F#`, `G`, `G#`, `A`, `A#`, `B`

## Minor
`Cm`, `C#m`, `Dm`, `D#m`, `Em`, `Fm`, `F#m`, `Gm`, `G#m`, `Am`, `A#m`, `Bm`

## Dominant 7th
`C7`, `C#7`, `D7`, `D#7`, `E7`, `F7`, `F#7`, `G7`, `G#7`, `A7`, `A#7`, `B7`

## Minor 7th
`Cm7`, `C#m7`, `Dm7`, `D#m7`, `Em7`, `Fm7`, `F#m7`, `Gm7`, `G#m7`, `Am7`, `A#m7`, `Bm7`

## Diminished

## Common progressions by genre

- **Pop (C major)**: C → G → Am → F
- **Rock (A minor)**: Am → F → C → G
- **Jazz (Dm)**: Dm7 → Gm7 → C7 → Fmaj7
- **Blues (E)**: E7 → A7 → E7 → B7
- **Ballad (G major)**: G → Em → C → D
- **Funk (Dm)**: Dm7 → Gm7 → A7 → Dm7
- **Bossa Nova (Cm)**: Cm7 → Fm7 → Dm7 → G7
